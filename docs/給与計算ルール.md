## 1. 時間外手当の集計ルール（A＝法定内残業／B＝法定外残業）

時間外は **A（法定内残業）** と **B（法定外残業）** の2区分で集計する。
判定は「日次」と「週次」の二段階で行う。

### 1-1. 日次の振り分け（基準：1日8時間を超えるか）

各日について、所定（契約）時間を超えた労働を次のように振り分ける。

- **A（法定内残業）** ＝ 所定時間を超え、かつ **1日8時間以内** の部分
- **B（法定外残業）** ＝ **1日8時間を超えた** 部分

計算式（その日の所定時間を `所定`、実労働を `実働` とする）：

```
A_日次 = max(0, min(実働, 8) − 所定)
B_日次 = max(0, 実働 − 8)
```

#### 例：木曜日の契約（所定）時間が4時間の場合
| 実労働 | A（法定内残業） | B（法定外残業） |
|--------|----------------|----------------|
| 5時間勤務 | A 1時間 | － |
| 9時間勤務 | A 4時間 | B 1時間 |

- 5時間：所定4hを1h超過、8h以内なので **A 1h**。
- 9時間：所定4h→8hまでの4hが **A**、8h超の1hが **B**。

### 1-2. 週次の上乗せ（基準：週合計40時間を超えるか）

日次の集計に加え、**週の合計労働時間が40時間を超えた分は「B」としてカウント**する。
すなわち、日次では「A（8時間以内）」と判定された残業であっても、その週の累計が40時間を超える部分は **B に振り替える**。

#### 例：月〜金で35時間労働し、土曜（所定4時間）に7時間労働した場合
1. 前5日（月〜金）の実績 35時間 に、土曜の所定 4時間 を足すと **39時間**。
2. 土曜の所定超過は 7−4＝**3時間**。1日の上限（8時間）以内だが、週40時間を基準に振り分ける。
3. 週40時間まで残り `40 − 39 = 1時間` → 土曜超過3hのうち **1hがA**。
4. 残り 2時間 は週40時間を超えるため **B**。

→ **土曜日の実績：A 1時間、B 2時間**

> ポイント：1日8時間以内であっても、週40時間超過分は B になる。週次判定が日次判定を上書きして B へ振り替える。

## 3. 非課税支給（交通費）

- 「非課税支給」項目は **主に時給職員の交通費** を指す。
- 雛形（テンプレート）に **「出勤日数」** を入力すると、各職員の **1日あたりの交通費単価** に基づき、**1ヶ月分の合計額を自動計算** する。

```
非課税支給（月額） = 交通費単価（1日あたり） × 出勤日数
```

## 4. 法定休日・日曜出勤の扱い

- 法定休日は就業規則に取り決めがないが、**日曜日** とみなして差し支えない。
- 日曜日に出勤する場合、**同じ週の別日に振替休日** を取得すれば、その日曜は
  **振替出勤（通常の出勤と同条件）** として給与計算が可能。
  - 振替休日が同週に取得されている → 当該日曜は割増なしの通常出勤扱い。
  - 振替休日がない日曜出勤は、別途（休日割増等の）扱いとなる点に注意。
